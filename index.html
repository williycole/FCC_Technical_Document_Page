Switch Statements

Special type of if statement that will allow us to compare ONE SPECIFIC value to a bunch of other certain values. This is shown below with break and case, case is our parameters that grade is checked with, the break says ok break out and check the next one default is used at the end like an else in case some accidentally enters the wrong thing

int main(){ char grade = 'B'; switch(grade){ case 'A': printf("You did great!"); break; case 'B': printf("You did alright!"); break; case 'C': printf("You did ok!"); break; case 'D': printf("You need improvement!"); break; case 'F': printf("You Failed!"); break; default : printf("Invalid Grade"); } return 0; }

int main(){ int index = 6; do{ printf("%d\n", index); index++; } while(index <= 5); /*what is in the () is the condition whats in {} is what is getting looked through to check the () conditon as long as whats in the () is is true the while loop will go through the code after it reaches greater than or equal to 5 it ends the loop*/ return 0; }
For Loops

For Loop: this lets us use an indexing variable, which tells us what iteration of the loop we are currently on. ForLoop condenses all the info in to one single loop, like putting multiple loops in one Loop container

int main(){ /*int i =1; while(i <=5){ printf("%d\n", i); i++; }*/ /*This shows the difference between while loops(above) vs for loops(below) and how a for loop is just a condensed while loop(better option possibly)*/ int luckyNumbers[] = {4, 8, 15, 16, 23, 42}; /*array indexes start at 0*/ int i; /*here you can see the while loop structure condensed into a for loop on line 23*/ for(i = 0; i< 6; i*3){ /*array indexes start at 0(4) so we start at 0 and go to till less than 6 since there are 5(ending at 42) elements in this array*/ printf("%d\n", luckyNumbers[i]); } /*for loops are vary handing, anything you can do with a while loop, you can do with a for loop */ return 0; }
