Here we use the printf function just to print our story, just like in the above example.

This will print to the Terminal There once was a young camper that was selft taught! However the \n will bring you to the next line and is not printed.

In the below code we see 3 strings contained in the "" quotation marks, you also see \n which brings you to the next line, a semi colon ; at the end of each printf();, and return which allows you to end and return the code.

int main(){ printf("Hello world!\n"); printf("There once was a young camper that was selft taught!\n"); printf("The camper's name was Jim.\n"); printf("He was 27 years old and went to school for Geology."); return 0; }

Below char and int are used, char deals with letters, where as int is for dealing with numbers.

Both char and int are used to create and define the variable names,%s is used as a place holder for characterName, and %d is used as a placeholder for characterAge.

This way all I have to do is change camper or 27 to something else and it will update when it prints where %s and %d. Data Types will be covered more in the next section.

int main(){ char characterName[] = 'camper'; int characterAge = 27; printf("There once was a man named %s\n", characterName); printf("he was %d years old.\n", characterAge); return 0; }
Data Types

int(integer): is for representing WHOLE numbers, no decimals. Here a variable is created age = to 40

double: is used for representing DECIMAL numbers, here the variable gpa = to 3.6 is created

char: is used for representing CHARACTERS, SINGLE characters are stored in single quotes ''

char name[] = "": how you can use char and double quotes to combine more than one character, or essentially STRING characters together to make an ARRAY

int main() { int age = 27; double gpa = 3.6; char grade = 'A'; char phrase[] = "freecodecamp"; return 0; }

Here is a list of format specifiers

%c a single character %s a string %hi short (signed) %hu short (unsigned) %Lf long double %lf double %n prints nothing %d a decimal integer (assumes base 10) %i a decimal integer (detects the base automatically) %o an octal (base 8) integer %x a hexadecimal (base 16) integer %p an address (or pointer) %f a floating point number for floats %u int unsigned decimal %e a floating point number in scientific notation %E a floating point number in scientific notation %% the % symbol
